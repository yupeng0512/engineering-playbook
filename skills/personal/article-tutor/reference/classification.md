# 笔记路径与分类体系

## 笔记路径规则

`learning-notes/{一级分类}/{二级分类}/{日期}-{标题}.md`

---


## 智能分类判断表

| 内容类型 | 一级分类 | 二级分类 |
|----------|----------|----------|
| AI 编程工具 | ai-tools | ai-ide / mcp / agent-skill |
| AI 架构设计 | ai-tools | agent-architecture |
| AI 工程实践 | ai-tools | vibe-engineering |
| AI 研究论文 | ai-research | deep-research / translation |
| 开源项目 | 根据领域判断 | 具体子分类 |
| 技术博客 | 根据主题判断 | 具体子分类 |

## 使用方式

Agent 在需要确定笔记分类时，读取此文件以：
1. 查看完整分类体系
2. 根据内容类型智能判断分类
3. 确定最终的笔记存储路径
