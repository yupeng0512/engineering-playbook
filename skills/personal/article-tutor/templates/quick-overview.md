# 快速概览卡片模板

## 知识类内容（技术博客/论文/书籍）

```markdown
## 📖 快速概览

| 项目 | 内容 |
|------|------|
| **标题** | {文章标题} |
| **作者** | {作者名称/背景} |
| **类型** | {技术博客/学术论文/书籍章节} |
| **核心主题** | {一句话概括} |
| **目标读者** | {适合什么人阅读} |
| **阅读价值** | ⭐⭐⭐⭐⭐ {简短评价} |

> 💡 **一句话总结**：{用最精炼的语言概括文章核心价值}

---

## 🎯 选择学习模式

| 模式 | 输出内容 | 适合场景 | 预计时间 |
|------|----------|----------|----------|
| **快速** | 概览 → 根节点笔记 | 时间有限，快速抓核心 | 2-3 分钟 |
| **标准** | 概览 → 根节点分析 → 行动清单 → 笔记 | 系统学习，全面理解 | 5-8 分钟 |
| **深度** | 全流程 + 苏格拉底问答式讲解 | 重要文章，深入研究 | 10-15 分钟 |

请回复 **"快速"**、**"标准"** 或 **"深度"**（默认：标准）
```
