## 国内首个"代际协同"微信小程序婚恋平台——让父母安心、子女放心
**天使轮融资 300-500 万元**

微信小程序 · 云开发 · 2026

---

<!-- _class: title-bullets -->

# 核心亮点

- **精准切口**：聚焦"父母为子女操心婚事"刚需场景，目标用户 2.4 亿单身人口背后的 **4 亿+** 焦虑父母
- **双端设计（行业首创）**：父母端（关怀模式 / 大字体）+ 子女端（年轻化交互），同一平台双角色协同
- **轻量部署**：基于微信小程序 + 云开发，零下载门槛，触达中老年用户无障碍
- **五级收入漏斗**：免费匹配引流 → 付费解锁联系方式 → VIP 会员 → 线下活动 → 情感增值服务
- **低成本高裂变**：父母群体天然强社交传播属性（广场舞群、家长群、亲友圈），K-factor **≥ 1.5**

# 五大增长驱动力

- **结婚率持续下降** → 父母焦虑升级，主动寻求线上渠道

- **代际协作需求** → 父母干预导致 **34%** 婚恋失败（行业调研），急需平台化沟通桥梁

# 单位经济模型（Unit Economics）

| 指标 | 数值 | 说明 |
|------|------|------|
| **CAC（获客成本）** | **15-30 元** | 微信生态裂变为主，远低于传统婚恋平台 200+ 元 |
| **LTV（用户生命周期价值）** | **80-150 元** | 付费用户平均生命周期 6-12 个月，ARPU 100-200 元/年 |
| **LTV / CAC** | **3-5x** | 健康区间（行业标准 ≥ 3） |
| **回收期** | **3-5 个月** | 付费用户首月即可回收 CAC |
| **毛利率** | **~85%** | 云开发成本极低，主要成本为人力和运营 |

> 收入增长飞轮：父母分享邀请 → 新用户注册 → 匹配池扩大 → 匹配成功率提升 → 口碑传播 → 付费意愿增强 → 收入增长

# 冷启动策略（M1-M3）

### 种子用户
创始团队亲友圈 + 本地社区 → 首批 **500-1,000** 用户

### 增长渠道（按优先级）
- **P0 父母社交裂变**：广场舞群、家长群分享，天然信任传递，CAC **5-10 元**
- **P1 微信公众号 + 视频号**：婚恋内容引流，天然高传播，CAC **10-20 元**
- **P2 本地社区合作**：居委会、妇联、工会渠道，CAC **15-25 元**
- **P3 短视频平台**：抖音/快手投放，父母向内容，CAC **30-50 元**
- **P4 区域代理/合伙人**：三四线城市本地化运营，分成模式（Phase 3）

### 增长飞轮
父母分享 → 新用户 → 匹配池扩大 → 成功率提升 → 口碑传播 → 更多父母分享
